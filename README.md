# dwQuizJson
Es un quiz de 5 preguntas el cual utiliza el localStorage del navegador para preservar la informacion desde el lado del cliente mientras se utiliza.
# Tecnologias
1. HTML
2. CSS
3. JS
4. Boostrap
# Funcionalidades
- Registro de usuario
- Inicio se sesion
- Ranking de mejores puntuaciones
